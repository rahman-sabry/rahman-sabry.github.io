About

I'm a backend developer at Tigerlab, where I build and maintain insurance web applications for enterprise clients using Python, Django, and Django REST Framework. Over three and a half years, my scope has expanded well beyond backend code — I now regularly work across containerised deployments, cloud infrastructure, and AI-powered features.

Before switching to software, I spent nearly three years at PERKESO as an investment analyst. A data pipeline I built in Excel VBA and PowerQuery M — which cut analyst processing time by over 50% — made it clear that building tools was what I actually wanted to do. I self-studied Python and transitioned into software development.

My early work was focused on Django features and debugging production issues. Over time I took on increasingly complex ownership — delivering the Renewals module for a client as the sole developer, redesigning a cancellation and refund system without disrupting live business flows, and resolving server-side database issues that could not be reproduced locally.

More recently I've been growing into DevOps and AI. I containerised an application for UAT using Docker and Kamal on AWS Graviton instances, managed a production S3 migration with zero client downtime, and built the foundational document embedding service that will power context-aware AI features in our platform. I'm AWS Certified and actively deepening my cloud architecture knowledge. I'm also expanding my frontend capability through React, with the goal of contributing meaningfully across the full stack.
